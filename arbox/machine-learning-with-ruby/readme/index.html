Awesome Machine Learning with Ruby [][ruby]

Curated List of Ruby Machine Learning Links and Resources

[Machine Learning][ml] is a field of [Computational Science][cs] - often nested under [AI][ai] research - with many practical applications due to the ability of resulting algorithms to systematically implement a specific solution without explicit programmer's instructions. Obviously many algorithms need a definition of [features][fe] to look at or a biggish [training set][ts] of data to derive the solution from.

This curated list comprises [awesome][awesome] libraries, data sources, tutorials and presentations about [Machine Learning][ml] utilizing the [Ruby][ruby] programming language.

A lot of useful resources on this list come from the development by [The Ruby Science Foundation][sciruby], our [contributors][contributors] and our own day to day work on various ML applications.

Every contribution is welcome! Add links through pull requests or create an issue to start a discussion.

Follow us on Twitter and please spread the word using the #RubyML hash tag!

Machine Learning Libraries

[Machine Learning][ml] algorithms in pure Ruby or written in other programming languages with appropriate bindings for Ruby.

Frameworks

  • wekastars65 - JRuby bindings for Weka, different ML algorithms implemented through Weka.
  • ai4rstars705 - Artificial Intelligence for Ruby.
  • classifier-rebornstars508 - General classifier module to allow Bayesian and other types of classifications. [dep: GLS]
  • scorubystars66 - Ruby scoring API for PMML (Predictive Model Markup Language).
  • rblearnstars1 - Feature Extraction and Crossvalidation library.
  • data_modelerstars1 - Model your data with machine learning. Ample test coverage, examples to start fast, complete documentation. Production ready since 1.0.0.
  • shogunstars2.8k - Polyfunctional and mature machine learning toolbox with Ruby bindings.
  • aws-sdk-machinelearningstars3.3k - Machine Learning API of the Amazon Web Services.
  • azure_mgmt_machine_learningstars263 - Machine Learning API of the Microsoft Azure.
  • machine_learning_workbenchstars13 - Growing machine learning framework written in pure Ruby, high performance computing using Numo, CUDA bindings through Cumostars86. Currently implementating neural networks, evolutionary strategies, vector quantization, and plenty of examples and utilities.
  • Deep NeuroEvolutionstars123 - Experimental setup based on the machine_learning_workbenchstars13 towards searching for deep neural networks (rather than training) using evolutionary algorithms. Applications to the OpenAI Gymstars24.1k using PyCallstars780.
  • rumalestars534 - Machine Learninig toolkit in Ruby with wide range of implemented algorithms (SVM, Logistic Regression, Linear Regression, Random Forest etc.) and interfaces similar to [Scikit-Learn][scikit] in Python.
  • epsstars544 - Bayesian Classification and Linear Regression with exports using PMML and an alternative backend using [GSL][gsl].

Neural networks

  • neural-net-rubystars109 - Neural network written in Ruby.
  • ruby-fannstars428 - Ruby bindings to the Fast Artificial Neural Network Library (FANN).
  • cerebrumstars35 - Experimental implementation for Artificial Neural Networks in Ruby.
  • tlearn-rbstars97 - Recurrent Neural Network library for Ruby.
  • brainsstars59 - Feed-forward neural networks for JRuby based on brainsstars0.
  • machine_learning_workbench - Framework including pure-Ruby implementation of both feed-forward and recurrent neural networks (fully connected). Training available using neuroevolution (Natural Evolution Strategies algorithms).
  • rannstars3 - Flexible Ruby ANN implementation with backprop (through-time, for recurrent nets), gradient checking, adagrad, and parallel batch execution.

Deep learning

  • tensor_streamstars499 - Ground-up and standalone reimplementation of TensorFlow for Ruby.
  • red-chainerstars95 - Deep learning framework for Ruby.
  • tensorflowstars829 - Ruby bindings for TensorFlow.
  • ruby-dnnstars34 - Simple deep learning for Ruby.
  • torch-rbstars347 - Ruby bindings for LibTorchstars48.1k using ricestars299.
  • mxnetstars49 - Ruby bindings for mxnet.

Kernel methods

  • rb-libsvmstars277 - Support Vector Machines with Ruby and the LIBSVM library. [dep: bundled]

Evolutionary algorithms

  • machine_learning_workbench - Framework including pure-Ruby implementations of Natural Evolution Strategy algorithms (black-box optimization), specifically Exponential NES (XNES), Separable NES (sNES), Block-Diagonal NES (BDNES) and more. Applications include neural network search/training (neuroevolution).
  • simple_gastars6 - Simplest Genetic Algorithms implementation in Ruby.

Bayesian methods

  • linnaeusstars37 - Redis-backed Bayesian classifier.
  • naive_bayesstars45 - Simple Naive Bayes classifier.
  • nbayesstars146 - Full-featured, Ruby implementation of Naive Bayes.

Decision trees

  • decisiontreestars1.3k - Decision Tree ID3 Algorithm in pure Ruby. [dep: GraphViz | post].

Clustering

  • flannstars1.6k - Fast Library for Approximate Nearest Neighbors. [flann]
  • kmeans-clustererstars79 - k-means clustering in Ruby.
  • k_meansstars113 - Attempting to build a fast, memory efficient K-Means program.
  • knnstars35 - Simple K Nearest Neighbour Algorithm.
  • annoy-rbstars15 - bindings for the Annoystars8.5k (Approximate Nearest Neighbors Oh Yeah).

Linear classifiers

  • liblinear-ruby-swigstars82 - Ruby interface to LIBLINEAR (much more efficient than LIBSVM for text classification).
  • liblinear-rubystars84 - Ruby interface to LIBLINEAR using SWIG.

Statistical models

  • rtimblstars5 - Memory based learners from the Timbl framework.
  • lda-rubystars132 - Ruby implementation of the LDA (Latent Dirichlet Allocation) for automatic Topic Modelling and Document Clustering.
  • maxent_string_classifierstars9 - JRuby maximum entropy classifier for string data, based on the OpenNLP Maxent framework.
  • omnicatstars10 - Generalized rack framework for text classifications.
  • omnicat-bayesstars30 - Naive Bayes text classification implementation as an OmniCat classifier strategy. [dep: bundled]

Gradient boosting

  • xgbooststars18 — Ruby bindings for XGBoost. [dep: XGBoost]
  • xgbstars58 — Ruby bindings for XGBoost. [dep: XGBoost]
  • lightgbmstars54 — Ruby bindings for LightGBM. [dep: LightGBM]

Applications of machine learning

  • phashionstars678 - Ruby wrapper around pHash, the perceptual hash library for detecting duplicate multimedia files. [ImageMagick | libjpeg]

Data structures

If you're going to implement your own ML algorithms you're probably interested in storing your feature sets efficiently. Look for appropriate data structuresstars552 in our [Data Science with Ruby][ds-with-ruby] list.
